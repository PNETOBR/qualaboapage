# QualABoa Page (React)

Landing page do app **QualABoa?** migrada para **React + Vite**, mantendo o mesmo visual e conteúdo da versão estática.

## Estrutura

```bash
.
├── public/
│  └── assets/
│     └── images/
│        ├── favicon.png
│        └── logo.png
├── src/
│  ├── components/
│  │  ├── Footer.jsx
│  │  ├── ForBusinesses.jsx
│  │  ├── Header.jsx
│  │  ├── Hero.jsx
│  │  └── HowItWorks.jsx
│  ├── App.jsx
│  ├── main.jsx
│  └── styles.css
├── index.html
├── package.json
└── vite.config.js
```

## Como rodar

```bash
npm install
npm run dev
```

Para build de produção:

```bash
npm run build
npm run preview
```

## Stack

- React 18
- Vite 5
- CSS puro
